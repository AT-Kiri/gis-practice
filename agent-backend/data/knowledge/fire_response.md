# 火灾扑救与人员疏散

## 一、火灾分类及灭火方法

依据 GB/T 4968-2008《火灾分类》标准：

| 类别 | 燃烧物 | 适用灭火剂 |
|------|--------|----------|
| A类 | 固体物质（木材、纸张、布料） | 水、泡沫、干粉 |
| B类 | 液体或可熔固体（汽油、酒精） | 泡沫、干粉、二氧化碳 |
| C类 | 气体（天然气、氢气） | 干粉、卤代烷，先切断气源 |
| D类 | 金属（钠、镁、钾） | 专用 D 类干粉、干砂 |
| E类 | 带电火灾 | 二氧化碳、卤代烷，先断电 |
| F类 | 烹饪油（动植物油脂） | 湿式化学灭火剂 |

## 二、城市建筑火灾扑救流程

标准作战流程按以下环节展开：

1. **报警**：119 接警中心受理，1 分钟内调派力量
2. **出动**：消防车 1 分钟内出库，城区 5 分钟到达现场
3. **侦察**：询问知情人、外部观察、内部侦察，掌握火情和被困人员
4. **灭火**：内攻为主、外部配合，水枪阵地步步推进
5. **搜救**：搜救组沿楼梯推进，逐层搜索房间，标记搜索结果
6. **通风**：破拆排烟、机械排烟，降低烟温和毒气浓度
7. **清理**：检查阴燃、防止复燃，保护火灾现场供调查

## 三、人员疏散原则

- **就近疏散**：使用最近的疏散楼梯，禁止使用普通电梯
- **有序撤离**：听从指挥，避免拥挤踩踏，老弱病残优先
- **方向正确**：向下疏散优先，火层以上向下、火层以下向上
- **防烟避难**：受烟气阻挡时进入避难层、防烟楼梯间等待救援

## 四、烟气危害与防护

火灾中 80% 以上死亡由烟气中毒窒息引起，烟气危害包括：

- **一氧化碳**：与血红蛋白结合导致缺氧窒息
- **氰化氢**：来源于装修材料燃烧，剧毒
- **高温灼伤**：烟气温度可达 300℃以上

防护措施：
- **低姿前进**：弯腰或匍匐，避开上层高温烟气
- **湿毛巾捂口鼻**：折叠 8 层可过滤 60% 烟尘
- **利用防烟分区**：关闭防火门，阻断烟气蔓延

## 五、森林火灾扑救方法

| 方法 | 适用场景 | 操作要点 |
|------|---------|---------|
| 直接扑打 | 弱度地表火 | 使用二号工具、风力灭火机沿火线扑打 |
| 以火攻火 | 强度高、难以直接扑救 | 在火前方点燃隔离带，烧除可燃物 |
| 开设隔离带 | 火势向居民区蔓延 | 伐木、清理灌木形成 5~10 米无植被带 |
| 航空灭火 | 火势大、地形复杂 | 直升机吊桶洒水、固定翼飞机洒化学药剂 |
| 人工增雨 | 大面积森林火灾 | 时机适合时实施人工降雨 |

森林火灾扑救必须严格遵守"三不打"原则：危险时段不打、危险地形不打、危险天气不打。

## 六、危险化学品火灾处置注意事项

危化品火灾扑救必须坚持"先控制、后消灭"原则：

- **查明性质**：通过 GHS 标签、运输单据确认危化品类别
- **禁止盲目射水**：遇水反应物质（如金属钠、电石）严禁用水
- **保持安全距离**：储罐火灾保持 2~3 倍罐径距离，防止爆炸冲击
- **冷却防爆**：对相邻储罐持续喷水冷却，防止热膨胀爆炸
- **撤离范围**：毒性气体火灾按下风向 3 公里范围疏散人员
- **个人防护**：进入火场人员必须穿戴正压式空气呼吸器、防化服

危化品火灾处置应由专业危化救援队伍主导，普通消防力量配合外围冷却和警戒。
