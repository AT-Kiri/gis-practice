# 综合应急预案框架

## 一、应急预案体系构成

依据《突发事件应对法》，我国应急预案体系由四个层级组成：

| 层级 | 编制主体 | 主要内容 |
|------|---------|---------|
| 总体预案 | 国务院 | 应急管理总纲，明确分类分级和工作机制 |
| 专项预案 | 国务院有关部门 | 针对特定灾种（地震、洪水、危化品等） |
| 部门预案 | 各部门 | 部门职责范围内的应急响应 |
| 地方预案 | 省/市/县/乡镇政府 | 辖区内突发事件应对，向下延伸至社区和企事业单位 |

预案应每 3 年修订一次，重大演练或实战后及时评估更新。

## 二、应急组织体系

```
应急指挥中心（党委政府主要领导）
        ↓
专项应急指挥部（主管部门牵头）
        ↓
现场指挥部（现场总指挥 + 副总指挥）
   ├── 综合协调组
   ├── 抢险救援组
   ├── 医疗救护组
   ├── 安全保卫组
   ├── 后勤保障组
   ├── 通信保障组
   ├── 新闻宣传组
   └── 善后处置组
```

现场指挥实行"属地为主、专业处置、分级负责"，指挥长由地方政府负责人担任，专业组接受双重指挥。

## 三、信息报告流程

| 阶段 | 时限 | 内容 |
|------|------|------|
| 首报 | 接报后 1 小时内 | 时间、地点、性质、伤亡初步情况 |
| 续报 | 每隔 2 小时或重大变化时 | 事态发展、处置进展、新增伤亡 |
| 终报 | 事件结束后 7 天内 | 完整经过、原因、责任、教训 |

报告原则："情况准确、要素完整、逐级上报"，特别重大事件可越级直报国务院。

## 四、应急资源储备

### 1. 物资清单
- 抢险类：挖掘机、生命探测仪、液压破拆工具
- 救生类：救生衣、抛投器、绳索、担架
- 防护类：防化服、空气呼吸器、防护口罩
- 生活类：帐篷、棉被、方便食品、饮用水（保障 3 天用量）

### 2. 装备配置
按"中央—省—市—县"四级储备，重点地区储备量上浮 50%。

### 3. 资金保障
- 应急储备金纳入财政预算，专款专用
- 重大灾害启动紧急拨款程序，24 小时内到位
- 鼓励社会捐赠和保险赔付作为补充

## 五、多部门协同机制

| 部门 | 主要职责 |
|------|---------|
| 消防 | 灭火、抢险救援、危化品处置 |
| 医疗 | 伤员救治、卫生防疫、心理援助 |
| 公安 | 现场警戒、交通管制、治安维护 |
| 交通 | 道路抢通、运力调度、绿色通道 |
| 通信 | 通信保障、应急基站、卫星电话 |
| 电力 | 电力抢修、应急发电、保电用户 |
| 气象 | 灾害预报、预警发布、气象保障 |
| 民政 | 受灾群众安置、灾情核查、社会捐助 |

协同机制通过联席会议、信息共享平台、联合演练落实，重大事件由指挥中心统一调度。

## 六、灾后恢复重建阶段划分

| 阶段 | 时间 | 主要任务 |
|------|------|---------|
| 抢险救灾 | 灾后 7~15 天 | 抢救生命、控制险情、保障基本生活 |
| 过渡安置 | 1~3 个月 | 临时安置、恢复公共服务、生产自救 |
| 恢复重建 | 1~3 年 | 重建住房、恢复产业、提升抗灾能力 |

恢复重建坚持"政府主导、群众主体、社会参与"原则，并融入防灾减灾理念，提升设防水平。

## 七、GIS 在应急管理中的应用

GIS 是现代应急管理的核心技术支撑，贯穿事前、事中、事后全过程：

- **风险评估**：叠加灾害历史数据、人口分布、建筑属性，绘制风险区划图
- **资源调度**：实时显示救援队伍、物资库、医疗点的位置，优化调度方案
- **路径规划**：基于路网数据和灾情路况，计算最优救援路径和疏散路径
- **态势展示**：在地图上动态展示灾情范围、扩散趋势、救援进展
- **避难场所分析**：基于服务半径和容量分析，优化避难场所布局
- **缓冲区分析**：评估灾害影响范围，划定警戒区
- **网络分析**：最短路径分析、服务区分析，支撑救援力量部署

通过 GIS 与物联网、遥感、大数据融合，实现应急管理的"可视化、智能化、精准化"。
