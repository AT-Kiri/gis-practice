# 医疗救援调度原则

## 一、伤员分类原则（START法）

批量伤员现场采用 START（Simple Triage and Rapid Treatment）快速分类法，依据呼吸、循环、意识三项指标评估，30 秒内完成一名伤员分类：

| 颜色标签 | 伤情等级 | 判断标准 | 处置原则 |
|---------|---------|---------|---------|
| 红色 | 危重 | 呼吸存在但需立即救治，循环不稳定或意识丧失 | 立即救治、优先转运 |
| 黄色 | 重伤 | 暂时稳定，但需手术或专科治疗，无法自行行走 | 延迟转运、1 小时内送医 |
| 绿色 | 轻伤 | 可自行行走，生命体征稳定 | 现场简单处理、自行就医 |
| 黑色 | 死亡 | 无呼吸、无脉搏，开放气道后仍无呼吸 | 现场封存、家属安抚 |

分类后伤员佩戴对应颜色标签，标签记录姓名、伤情、处置时间。

## 二、伤员转运优先级

- **红色（立即转运）**：5 分钟内安排救护车转运至三级医院
- **黄色（延迟转运）**：30 分钟内转运至二级及以上医院
- **绿色（可自行行走）**：集中运送至社区医院或临时医疗点观察
- **黑色（死亡）**：原地封存，由殡葬部门统一处理，不得混入伤员转运

转运实行"检伤—急救—转运—分流"闭环管理，避免"扎堆"涌入同一医院造成二次灾害。

## 三、医疗资源调度

### 1. 就近原则
优先调度距离事故现场 5 公里内的医疗机构，超过 30 分钟车程视为不可及。

### 2. 专科对口
- 颅脑外伤 → 神经外科中心
- 大面积烧伤 → 烧伤专科医院
- 多发伤 → 创伤中心
- 复合中毒 → 中毒救治中心

### 3. 批量伤员分流
- 30% 重伤员送三级甲等医院
- 50% 中度伤员送二级医院
- 20% 轻伤员送社区医院或留观
- 同一医院接收重伤员不超过 20 名，避免超载

## 四、现场急救基本措施

按"止血—包扎—固定—搬运—复苏"顺序处置：

1. **止血**：直接压迫止血为主，必要时使用止血带（标记时间，每 1 小时松解 1 分钟）
2. **包扎**：使用三角巾或绷带，暴露伤口，禁止回纳脱出肠管
3. **固定**：骨折部位就地固定，固定范围超过上下关节
4. **搬运**：脊柱伤员使用硬质担架整体翻身，避免二次损伤
5. **心肺复苏**：CAB 顺序——胸外按压（30 次，深度 5~6 cm）→ 开放气道 → 人工呼吸（2 次），按压频率 100~120 次/分

## 五、临时医疗点设置要求

| 要素 | 要求 |
|------|------|
| 位置 | 距事故现场 50~200 米的上风方向 |
| 交通 | 距主干道近，救护车可直达 |
| 水源 | 30 米内有清洁水源 |
| 面积 | 每名伤员不少于 4 平方米 |
| 分区 | 检伤区、抢救区、稳定区、转运区、停尸区 |
| 标识 | 红十字旗夜间发光，距 100 米可见 |

## 六、心理援助与危机干预

灾害后 30%~40% 幸存者会出现急性应激反应，72 小时内启动心理干预：

- **早期介入**：陪伴倾听，不打断、不评判，提供安全感
- **重点人群**：直接受害者、救援人员、遇难者家属
- **危机干预六步法**：明确问题→保证安全→提供支持→评估方案→制定计划→获得承诺
- **转介机制**：识别 PTSD 高危者，转介精神专科长期随访

心理援助人员应统一佩戴标识，避免反复询问创伤经历造成二次伤害。
