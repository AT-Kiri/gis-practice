# 数据大屏模块 - 实现清单

## 优先级说明

| 标记 | 含义 |
|------|------|
| P0 | 必须完成，阻塞其他任务 |
| P1 | 核心功能，必须完成 |
| P2 | 重要功能，建议完成 |
| P3 | 增强功能，时间允许时完成 |

## 任务列表

### Task 1：路由与导航入口 [P0]

| 属性 | 内容 |
|------|------|
| 依赖 | 无 |
| 文件 | `router/index.js`、`App.vue` |
| 描述 | 新增 `/data-dashboard` 路由（懒加载），顶部导航栏新增"数据大屏"按钮 |
| 验证 | 点击导航按钮跳转到数据大屏页面，按钮高亮，可正常返回其他页面 |

### Task 2：模拟数据生成工具 [P0]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 1 |
| 文件 | `utils/mockData.js` |
| 描述 | 创建模拟数据生成模块，包含全部京津冀县区的灾害数据、救援资源、物资点、气象数据 |
| 验证 | 各生成函数返回结构正确的数据，覆盖所有县区 |

### Task 3：DataDashboardView 页面容器 [P0]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 1, Task 2 |
| 文件 | `views/DataDashboardView.vue` |
| 描述 | 页面主容器：半屏面板式布局，左侧地图区域（SmMapViewer），右侧上下两个面板区域 |
| 验证 | 页面正常渲染，地图和面板布局正确 |

### Task 4：DashboardMap 分级地图组件 [P1]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 3 |
| 文件 | `components/dashboard/DashboardMap.vue` |
| 描述 | 加载 iServer 行政区划 GeoJSON + mock 灾害数据，按等级渲染填充颜色；支持悬停 tooltip 和点击选中 |
| 验证 | 各县区按灾害等级显示对应颜色，鼠标悬停显示 tooltip，点击弹出详情 |

### Task 5：DisasterDetailPanel 灾害详情面板 [P1]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 3, Task 4 |
| 文件 | `components/dashboard/DisasterDetailPanel.vue` |
| 描述 | 展示选中县区的灾害详情（地区、类型、等级、救援/受灾人员、备注），含"缓冲区分析"联动按钮 |
| 验证 | 点击县区后右侧面板正确显示详情，未选中时显示提示信息 |

### Task 6：BufferAnalysisModal 缓冲区分析弹窗 [P1]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 5 |
| 文件 | `components/dashboard/BufferAnalysisModal.vue` |
| 描述 | 以选中县区为中心做缓冲区分析，可调节半径（1-50km），展示附近救援人员和物资点列表，地图渲染缓冲区和标记点 |
| 验证 | 弹窗打开后正确显示缓冲区，调整半径后数据更新，关闭弹窗后清除地图标记 |

### Task 7：RoutePlanningModal 路径规划弹窗 [P2]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 6 |
| 文件 | `components/dashboard/RoutePlanningModal.vue` |
| 描述 | 点击救援按钮后，计算从救援点到受灾县区的最优路径，弹窗展示距离和时间，地图高亮路径 |
| 验证 | 路径规划正常显示，若网络分析失败则降级显示直线路径 |

### Task 8：WeatherPanel 气象监控面板 [P1]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 3, Task 2 |
| 文件 | `components/dashboard/WeatherPanel.vue` |
| 描述 | 以表格展示多地区气象数据（风险等级、大风、降雨、地震强度、受困/物资），超阈值标红，底部显示前后几天天气趋势 |
| 验证 | 数据正确展示，阈值标红有效，表格可排序，天气预报正常显示 |

### Task 9：地图缓冲区圆圈与资源点渲染 [P2]

| 属性 | 内容 |
|------|------|
| 依赖 | Task 6 |
| 文件 | `components/dashboard/BufferAnalysisModal.vue`（可内联实现） |
| 描述 | 在地图上绘制半透明缓冲区圆圈 + 救援点（蓝色标记） + 物资点（橙色标记） |
| 验证 | 缓冲区和标记点正确显示，随半径调节刷新，弹窗关闭后清除 |

### Task 10：整体样式与交互打磨 [P3]

| 属性 | 内容 |

| 验证 | 视觉风格统一，交互流畅，无明显的 UI 问题 |
